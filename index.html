Getting Started

Please!! Read carefully before applying on your device. This time I have updated my device's BIOS to the latest version. The latest version is currently at X441UVK.324. Good luck!!.

1. Using Windows Press Windows + R on Keyboard   ›   write "dxdiag" and enter, look at BIOS.
2. Using BIOS Press F2 on Keyboard when boot   ›   Look at BIOS Version.
3. If the bios version matches the specified requirements, you can start the next step.

Fix Wifi

By default the AR9565 Wifi card now working after post installation. That's because you have to fix it manually, atleast if you want to have it work properly.

1. Download the AirPortAtheros40.kext file for the AR9565 model from my repository and download Kextbeast 2.0.2 from tonymacX86.com or internet.
2. Extract AirPortAtheros40.kext.zip and place AirPortAtheros40.kext on the desktop.
3. Run Kextbeast 2.0.2 and select the installation location at Library/Extensions.
4. Reboot.

Fix FN Keys

Here I'm going to show you how can fix the FN Keys button Asus Laptop on Hackintosh.

1. Download the AsusSMC package from the following repository https://github.com/hieplpvip/AsusSMC/releases.
1. Extract package and run the script install_daemon.sh .
1. Reboot after complete.

Fix Audio

Here is how you can fix audio problems:

1. Download the CodecCommander.kext file from my repository and Kextbeast 2.0.2 from tonymacX86.com or internet.
2. Extract CodecCommander.kext.zip and place CodecCommander.kext on the desktop.
3. Run Kextbeast 2.0.2 and select the installation location at Library/Extensions.
4.After the above steps are complete, Download the ALCPlugFix for ALC255 from the following repository https://github.com/black-dragon74/ALCPlugFix/releases.
5. Run the script and reboot after complete all.
